# Bingo em Casa

Projeto estático para brincar de bingo 90 bolas sem cadastro e sem backend.

## Páginas

- `index.html`: tela inicial para escolher entre sorteador e cartela.
- `sorteador.html`: sorteador de números de 1 a 90, com histórico, persistência local e modo automático de 7 segundos.
- `cartela.html`: cartela digital 90 bolas, responsiva, persistente e com modo retrato/paisagem.

## Tecnologias

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages

## Como publicar no GitHub Pages

Suba os três arquivos na raiz do repositório e ative o GitHub Pages em `Settings > Pages`, usando a branch `main` e a pasta `/root`.

## Observação

A persistência é local por navegador/dispositivo. Se o usuário limpar os dados do navegador, as informações salvas localmente poderão ser apagadas.
